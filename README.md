# Snow Rider: Anime Edition

## Overview

Snow Rider: Anime Edition is a 2D snowboarding game built with Unity where players choose their favorite anime character and ride through challenging snowy tracks. Perform flips to earn points, collect power-ups to boost your abilities, avoid obstacles, and reach the legendary Kurama at the finish line to complete the level.

## Features

* Character Selection System

  * Choose between Naruto and Goku before starting the game.

* Physics-Based Snowboarding

  * Smooth movement and controls powered by Unity's 2D physics system.

* Flip Scoring System

  * Perform aerial flips to earn bonus score points.

* Power-Up Collection

  * Collect power-ups that enhance player abilities such as speed and control.

* Collision Detection

  * Avoid dangerous collisions while navigating the track.

* Finish Line Objective

  * Reach Kurama at the end of the level to win the game.

* Particle Effects

  * Snow trail effects for a more immersive gameplay experience.

## Controls

* Left Arrow / A - Rotate Left
* Right Arrow / D - Rotate Right
* Up Arrow / W - Boost Speed

## Technologies Used

* Unity 6
* C#
* Unity Input System
* 2D Physics Engine
* Particle System

## Gameplay

1. Select Naruto or Goku.
2. Ride through the snowy track.
3. Perform flips to gain score points.
4. Collect power-ups to improve performance.
5. Avoid crashes.
6. Reach Kurama to complete the level.
